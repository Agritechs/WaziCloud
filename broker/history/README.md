Broker History API
==================

This API allows to retrieve the history of sensor measurements.

Install
-------

Build the image:

```
$ docker build -t waziup/sth-comet .
$ docker run -t --net=host waziup/sth-comet

```
Or simply run

```
$ docker-compose up

```
